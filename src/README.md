# 🔐 Gerador de Senhas em C

Projeto desenvolvido em linguagem C para gerar senhas aleatórias de acordo com as preferências do usuário.

## Funcionalidades

- Escolha do tamanho da senha
- Letras maiúsculas
- Letras minúsculas
- Números
- Caracteres especiais
- Geração de múltiplas senhas sem reiniciar o programa
- Validação do tamanho da senha
